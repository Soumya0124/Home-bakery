# Artisan Bakery - Full-Stack E-commerce App

Live Frontend: https://harmonious-fenglisu-f58de2.netlify.app  
Live Backend API: https://home-bakery-tm22.onrender.com/api/products  

Features:
- Responsive React UI with Vite
- Product listing, add to cart, quantity controls
- LocalStorage cart persistence
- Express backend with /api/products & /api/checkout
- Real checkout flow (logs order on server)

Tech Stack:
- Frontend: React, Vite, Axios
- Backend: Node.js, Express, CORS
- Deployment: Netlify (frontend), Render (backend)
